Meus Projetos

DsTasty

O projeto foi uma oportunidade para aprofundar meus conhecimentos em HTML e CSS. Durante o desenvolvimento, organizei o conteúdo em seções distintas com tags semânticas, melhorando a acessibilidade e a compreensão do código. No CSS, explorei o uso de classes e containers, aprendendo a importância de uma boa estrutura no layout. Também apliquei propriedades como display, align-items e justify-content para criar layouts responsivos e esteticamente agradáveis. Além disso, utilizei a API de Fontes do Google para personalizar a tipografia, contribuindo para uma apresentação visual moderna e atraente.

Jogo da Forca

O projeto "Jogo da Forca" foi uma excelente oportunidade para me aprofundar no uso de Java e aprimorar minhas habilidades em programação orientada a objetos. Durante o desenvolvimento, organizei o código em classes, garantindo uma separação clara entre as responsabilidades de cada componente, como a lógica do jogo, a leitura de arquivos e o controle de exceções. Explorei o uso de ferramentas como BufferedReader e Scanner para manipulação de dados, além de Random para a escolha aleatória de palavras, proporcionando uma experiência dinâmica ao usuário. Também implementei tratamento de exceções, garantindo a robustez do sistema e prevenindo erros inesperados.
